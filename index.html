Common CSS Properties

Font Properties
Property Explanation
Font Family Specifies the font to be used
Font Style Specifies which style of font to be used. Default is normal, others include italics and oblique
Font Size Specifies the size that text will be displayed in units such as pixels or a percentage
Font Weight Specifies the boldness of a font. Examples include bold and lighter
Element Organization
Property Explanation
Display Sets whether an element is treated as a block or inline element. Examples include block, inline, and inline-block
Position Sets how an element is positioned in a document. Examples include static, relative, absolute, and sticky
Z-Index Sets the Z-order of a positioned element. Value is a number, default is 0. This can be used to allow one element to display "on top" or "below" another element
